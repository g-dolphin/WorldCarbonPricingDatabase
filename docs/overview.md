# Overview

This repository is the public distribution and transparency repository for the World Carbon Pricing Database (WCPD).

## What is included
1. Released dataset files in `_dataset/data`.
2. Data-point source citations in `_dataset/sources`.
3. Public documentation for users of the released data.

## Note on production workflow
The production workflow used to curate raw inputs, ingest sources, and compile releases is maintained in a separate private repository.
